# ICML 2025 论文仓库合集 / ICML 2025 Paper Repositories Collection

本文档收集了GitHub上汇总ICML 2025论文的仓库。这些仓库提供了论文列表、代码实现以及相关资源。

This document collects repositories on GitHub that compile ICML 2025 papers. These repositories provide paper lists, code implementations, and related resources.

## 主要仓库 / Main Repositories

### 1. ICML2025-Papers-with-Code
- **仓库地址 / Repository**: [https://github.com/yinizhilian/ICML2025-Papers-with-Code](https://github.com/yinizhilian/ICML2025-Papers-with-Code)
- **描述 / Description**: 历年ICML论文和开源项目合集，包含ICML2021、ICML2022、ICML2023、ICML2024、ICML2025
- **Stars**: ⭐ 35+
- **特点 / Features**: 全面收录历年ICML论文，包含多年度论文集合

### 2. ICML-2025-Papers (DmitryRyumin)
- **仓库地址 / Repository**: [https://github.com/DmitryRyumin/ICML-2025-Papers](https://github.com/DmitryRyumin/ICML-2025-Papers)
- **描述 / Description**: ICML 2025 Papers: Dive into cutting-edge research from the premier machine learning conference. Stay current with breakthroughs in deep learning, generative AI, optimization, reinforcement learning, and beyond. Code implementations included.
- **Stars**: ⭐ 17+
- **特点 / Features**: 
  - 包含代码实现
  - 涵盖深度学习、生成式AI、优化、强化学习等多个主题
  - 详细的主题标签分类
- **Topics**: ai-research, deep-learning, diffusion-models, generative-ai, graph-learning, icml, icml-2025, machine-learning, multimodal-ai, optimization, reinforcement-learning, reinforcement-learning-algorithms

## 如何选择合适的仓库 / How to Choose the Right Repository

1. **综合性需求**: 推荐使用 `yinizhilian/ICML2025-Papers-with-Code` 或 `DmitryRyumin/ICML-2025-Papers`，它们提供了全面的论文收录
2. **代码实现需求**: `DmitryRyumin/ICML-2025-Papers` 明确标注包含代码实现
3. **特定领域**: 
   - 机器人学 → `icml2025-robotics-papers`
   - 扩散模型 → `Diffusion-Papers-ICML-2025`
